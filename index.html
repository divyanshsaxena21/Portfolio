Projects

Face Mask Detection

This project involves a Convolutional Neural Network (CNN) designed to detect face mask usage. The model features two convolutional layers that extract essential image features and two dense layers that classify the mask-wearing status. By processing and analyzing visual data through this structure, the CNN efficiently determines whether individuals are wearing masks.

Fake News Prediction

This project involves classifying news articles as "fake" or "real" using text preprocessing and machine learning. We start by stemming words to their base forms, then use vectorization methods like Bag of Words or TF-IDF to convert text into numerical features. These features are fed into a logistic regression model to predict the authenticity of news articles.

Cat-Vs-Dog Classification

This project involves building a Convolutional Neural Network (CNN) to classify images as "dog" or "cat." We preprocess a dataset of images, design a CNN to learn features through convolutional and pooling layers, and train it to differentiate between the two classes. The model is then evaluated for accuracy and used to classify new images.

Virtual Quiz Game

Vi_Quiz is an interactive virtual quiz game developed during my 3rd semester of B.Tech. Using Python, OpenCV, and Mediapipe, the game leverages computer vision for engaging user interactions, such as gesture-based inputs and real-time visual effects. MySQL is utilized for managing and storing quiz data, including questions, answers, and user scores. This integration ensures a dynamic and responsive quiz experience, combining advanced technology with interactive gameplay..
